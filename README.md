Веб-застосунок для створення списку справ.

    Простий клієнтський застосунок для створення списку справ, розроблений за допомогою JavaScript.
Застосунок дозволяє користувачам створювати, виконувати, видаляти та зберігати завдання за допомогою браузера localStorage.

## Функції
- Додавання нових завдань
- Позначення завдань як виконаних
- Видалення завдань
- Збереження даних у localStorage
- Очищення всіх завдань

## Структура проекту
- index.html – Розмітка інтерфейсу користувача
- style.css – Стиль
- script.js – Логіка застосунку
- script.test.js – Модульні тести

## Тестування
Модульні тести реалізовано за допомогою Jest.
Охоплені функції:
- addTask
- deleteTask
- toggleTaskCompletion
- generateId
- saveToStorage
